# Second-Hand Marketplace - Frontend

This is the **frontend** of the Second-Hand Marketplace project, built using **React**. It provides a user-friendly interface for buyers, sellers, and admins to interact with the system.

---

## Live Website
- [Live Website Link](https://secondhand-marketplace-fullstack-proj.netlify.app/)

## GitHub Repository
- [Frontend Code](https://github.com/Sadia437/Second-Hand-Marketplace-Frontend.git)

## Features

- User authentication (Login / Register)
- Admin panel and dashboard
- User dashboard
- Make admin functionality
- Table display inside dashboards
- Fetch data from backend API
- Query & filtering functionality
- React hooks
- Form handling and validation
- Context API for global state
- Custom hooks for reusable logic
- Responsive UI design

---

## Technologies Used

- React
- React Router
- Context API
- Tailwind CSS (if used)
- Axios / Fetch for API calls
- React Hook Form (for forms)
- Vercel / Netlify for deployment
